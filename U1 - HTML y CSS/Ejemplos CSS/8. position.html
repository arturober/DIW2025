Ejemplo de position: fixed

Haz scroll hacia abajo para ver cómo la caja verde se mantiene en su sitio.

El elemento con position: fixed es retirado del flujo normal del documento. Esto significa que no ocupa espacio y otros elementos se comportarán como si no estuviera ahí.

Por eso, se posiciona relativo a la ventana gráfica del navegador (el viewport).
